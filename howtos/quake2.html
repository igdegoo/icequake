Quake II Guide

Last updated: 8/8/2018 by yoshi295295

If you do not have a copy, you can get it used from eBay or goodwills or you can download a pre-configured copy with the CD soundtrack from here. The copy provided by me uses Yamagi Quake 2, which is my recommended source port.

What if you prefer to install the game yourself?

If you have a physical Quake 2 CD or a ISO you want to install from, you will need a source port to play, since when you try to install the game on Windows XP and later, you will get a error.

There is a variety of source ports, but for this guide, we will focus on Yamagi Quake 2. Download it from here. Then put the file in a empty folder and run it. The file is a self-extracting archive. Extract the files to the folder, then delete the following files:

  • 3.20_Changes.txt
  • quake2.exe
  • ref_gl.dll
  • ref_soft.dll
  • baseq2/gamex86.dll
  • ctf/ctf2.ico
  • ctf/gamex86.dll
  • ctf/readme.txt
  • ctf/server.cfg
  • xatrix/gamex86.dll
  • rogue/gamex86.dll

After you have deleted the files, copy pak0.pak and the video folder from your CD/ISO to the baseq2 folder of the patch.

If you have mission packs, copy the pak0.pak and the video folder of each one into their respective folder:

The Reckoning: xatrix

Ground Zero: rogue

If you want to have the CD soundtrack for the basegame you have 2 options.

1. Download it from here. This is a good option if you don't want to rip the songs yourself or you have the game in ISO form.

2. Rip the soundtrack from your CD using CDex. Be sure to set the ripped song filetype as OGG Vorbis. You can't rip track 1, since the track is used for data. After the songs are ripped, name the songs 02.ogg for track 02, 03.ogg for track 03 and so on until you reach track 11. Then put the songs into the music folder inside the baseq2 folder.

If you want to rip the songs from the mission pack CDs, you can do option 2 but instead of putting the songs inside the baseq2/music folder you put them inside the music folder of the respective folder for the mission packs.

The Reckoning: xatrix/music

Ground Zero: rogue/music

Once you are done with all that, go to https://www.yamagi.org/quake2 and scroll down to the Downloads section.

Then click on the Precompiled Windows binaries link.

Extract the zip file you have just downloaded and move all of the files inside the patch. Start yquake2.exe and you should be playing Quake 2.
